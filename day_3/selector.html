Selector

selector is used to select elements in the html

selector is used to apply css to the selected elements

Types of selector

  1. element Selector
  2. id Selector
  3. Class selector
  4. Group Selector
  5. attribute Selector
  6. Universal Selector
  7. Descendent Selector
  8. Child Selector
  9. adjacent Sibling selector
  10. general sibling selector

element selector

element selector is used to select the elements based on the tag name

class selector

  • class selector is used to select the elements based on the class name
  • class selector not to be unique
  • we have to use class selector with .className for css styles

id selector

  • id selector is used to select elements based on id name
  • id selector must be unique
  • we can use id selector with #idName for CSS style

Rules :

  1. class/id name should have alphabets and numbers(-,_)
